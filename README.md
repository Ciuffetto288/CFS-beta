### Nota Importante

Il sistema CFS è progettato per evolversi verso funzionalità di analisi e automazione finanziaria basate su AI, incluse possibili funzionalità di suggerimento e gestione autonoma del capitale.

Tuttavia, l’utilizzo del software comporta le seguenti condizioni obbligatorie:

#### Responsabilità e Limitazioni

Gli sviluppatori, collaboratori e contributori del progetto non si assumono alcuna responsabilità per:

- Perdite finanziarie dirette o indirette derivanti dall’utilizzo del software
- Decisioni di investimento prese in modo autonomo o assistito dal sistema
- Errori di calcolo, predizione o analisi dei dati di mercato
- Malfunzionamenti delle integrazioni API con exchange o servizi esterni
- Utilizzo improprio del software da parte dell’utente
- Conseguenze fiscali, legali o amministrative derivanti dall’uso dei report generati

#### Autonomia del Sistema

Le funzionalità AI, ove implementate, possono operare in modo autonomo o semi-autonomo nell’analisi dei dati e nella proposta o esecuzione di operazioni finanziarie.

L’utente riconosce e accetta che:

- Il sistema può prendere decisioni automatizzate basate su modelli statistici e predittivi
- Tali decisioni non costituiscono consulenza finanziaria professionale
- Il comportamento del sistema può variare in base a dati incompleti o non aggiornati
- L’utente rimane l’unico responsabile dell’attivazione, configurazione e utilizzo del sistema

#### Uso Consapevole

Il software è destinato esclusivamente a utenti consapevoli dei rischi associati ai mercati finanziari e alle criptovalute.

Si raccomanda sempre la consulenza di un professionista qualificato prima di prendere decisioni finanziarie o fiscali.

#### Nessuna Garanzia

Il software è fornito "così com’è", senza alcuna garanzia di accuratezza, affidabilità o idoneità a scopi specifici.
